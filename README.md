# Python Quiz Application

## Description

A simple command-line quiz application developed in Python. The application asks multiple-choice questions and calculates the final score.

## Features

- Multiple-choice questions
- Automatic score calculation
- Displays final percentage
- User-friendly interface

## Technologies Used

- Python 3

## How to Run

1. Download the project
2. Open terminal
3. Run:

```bash
python quiz_app.py
```
